#MAPS

・「ローカル情報を共有」できる旅アプリ。

＃背景

・個人的に自転車一人旅が好きなので、旅好きな人達がガイドブックに乗っていないローカル情報を共有でき、コロナ禍で移動が制限された中でもオンラインで旅を追体験できるアプリを制作しました。
